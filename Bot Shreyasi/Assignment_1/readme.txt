🌦️ Weather Data Management
Overview
This Python script allows you to manage and analyze weather data records. You can input temperature and humidity data for different cities and get the average temperature and humidity for each city.

Features
Input Records: Add weather data for various cities.
Calculate Averages: Compute average temperature and humidity for each city.
Interactive: User-friendly prompts for data entry.
How It Works
Input Records: You will be prompted to enter the number of records you want to input. For each record, you'll need to provide the city name, temperature, and humidity.
Weather Calculation: The script calculates the average temperature and humidity for each city based on the provided records.
Results: The averages are displayed in a dictionary format, with each city’s average temperature and humidity.
